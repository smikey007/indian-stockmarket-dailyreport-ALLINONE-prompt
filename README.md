# India Stock Market Daily Report Prompt

A professional swing trader intelligence prompt for the Indian stock market.
Built for use with Claude AI (claude.ai or Claude Code).

## How to use
1. Copy the prompt from `market-prompt.md`
2. Paste it into Claude and run
3. Or save it as a slash command in Claude Code using `/market`

## What it covers
- Live Nifty / Bank Nifty / Sensex + full sector map
- Breaking catalysts + corporate announcements
- FII/DII provisional flows
- Earnings calendar (next 14 days)
- Institutional brokerage coverage with post-news pump %
- Unusual volume + options OI intelligence
- Promoter / insider SAST disclosures
- Social signals, SEBI actions, print media calls
- Signal Convergence Watchlist (scored 1–7)
